# 情侣日常小程序
在[UxxHans Rainbow Cats](https://github.com/UxxHans/Rainbow-Cats-Personal-WeChat-MiniProgram)基础上进行二次开发,实现了任务布置,积分商城功能,增加相册记事本的功能，增加纪念日倒计时功能.  
1.突破二十条任务上限.  
2.允许上传图片.
